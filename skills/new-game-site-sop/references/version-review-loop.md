# 版本复查闭环

## 稳定性分组

- STATIC：世界观、固定地图、静态 NPC
- SEMI-STABLE：材料、配方、一般掉落
- VOLATILE：Build、Meta、Tier List、平衡性

## 触发条件

- 官方 patch notes 更新
- 开发者公告
- 社区共识明显变化
- 数据挖掘结果变化
- 实测结果和现页结论冲突

## 默认复查周期

- STATIC：180 天
- SEMI-STABLE：30 到 60 天
- VOLATILE：14 天

## 处理方式

- 结论过期时，标记为“待验证”
- 更新“已确认 / 未确认 / 已过期”状态
- 不要继续把旧版本结论当作确定内容写
