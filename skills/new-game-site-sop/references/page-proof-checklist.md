# 页面证据检查清单

在任何新游戏攻略页上线前，都先过这一份清单。

## 最低发布门槛

一页内容至少满足下面 5 项里的 3 项：

- 原创截图
- 自制对比表
- 自制路线图或决策流
- 明确推荐结论和理由
- 明确写出不确定项或例外条件

## 发布前问题

1. 这页到底在回答玩家什么问题？
2. 这页里有什么内容是通用 AI 总结做不出来的？
3. 哪些结论来自官方信息？
4. 哪些结论来自社区观察？
5. 哪些结论目前还没有完全确认？

## 弱信号

如果一页主要由这些东西构成，先别发：

- 改写 Wiki 文本
- 没有理由的“best”判断
- FAQ 灌水
- 重复型开头文案
- 覆盖面很广但证据很薄

## 强信号

如果一页包含这些内容，可以优先上线：

- 截图支撑的 walkthrough
- 带版本意识的 patch 说明
- 真实 trade-off 分析
- “什么时候不要这样做”的反例说明
- 能直接帮用户缩短决策时间的对比表
