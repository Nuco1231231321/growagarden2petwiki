# SERP 调研矩阵

用这张矩阵做游戏关键词调研，不要只看一页 Google 结果。

## 调研层

### 1. 官方层
- 官网
- patch notes
- 开发者帖子
- 商店页

### 2. 排名层
- 当前 SERP 前排页面
- 标题模式
- 页面类型模式
- 缺失内容

### 3. 社区层
- Reddit
- Steam 讨论
- Discord 摘要
- 论坛

### 4. 创作者层
- YouTube 视频
- 视频转录
- 反复出现的建议

### 5. Query 扩展层
- 同义词
- 缩写
- 错拼
- `best`
- `worth it`
- `how to`
- `where`
- `build`
- `unlock`
- `fix`

### 6. 证据层
- 哪些能被官方确认
- 哪些只是社区共识
- 哪些仍然不清楚

## 每层要提取什么

每一层都提取：

- 最强用户需求
- 最强重复主张
- 证据质量
- 哪部分没讲清楚

## 给草稿阶段怎么用

用这张矩阵决定：

- 页面类型
- section 顺序
- 哪些表格值得放
- 哪些地方必须留人工证据槽位
